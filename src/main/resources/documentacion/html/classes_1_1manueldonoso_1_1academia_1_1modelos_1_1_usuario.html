Detailed Description

Representa un usuario en el sistema con información básica de contacto y autenticación.

La clase Usuario contiene los datos de un usuario, incluyendo su nombre, apellidos, dirección, teléfono, contraseña, tipo de usuario, una foto en formato de arreglo de bytes, y fechas de alta y baja en el sistema. El tipo de usuario está limitado a los valores "ADMINISTRADOR", "PROFESOR" o "ALUMNO".
