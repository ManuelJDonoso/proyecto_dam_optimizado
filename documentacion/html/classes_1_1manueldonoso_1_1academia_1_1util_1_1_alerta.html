Detailed Description

Clase utilitaria para mostrar diferentes tipos de alertas en la aplicación. Proporciona métodos para mostrar alertas de error y confirmación con un estilo personalizado.

Muestra una alerta de confirmación y devuelve la respuesta del usuario.

Parameters
tituloEl título de la ventana de alerta.
barraEl texto de la barra de encabezado.
mensajeEl contenido del mensaje a mostrar.
Returns
true si el usuario selecciona "OK", false en caso contrario.

Este método utiliza la clase Alert de JavaFX con el tipo AlertType.CONFIRMATION. Personaliza la ventana eliminando la barra de título y ajustando la opacidad.

Ejemplo de uso:

    boolean confirmado = Alerta.Confirmacion("Salir", "Confirmación requerida", "¿Está seguro de salir?");
    if (confirmado) {
        System.exit(0);
    }
