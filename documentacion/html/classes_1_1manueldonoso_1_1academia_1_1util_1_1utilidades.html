static void es.manueldonoso.academia.util.utilidades.log ( String message)
static

Imprime un mensaje de log junto con información sobre la clase, el método y la línea desde donde fue llamado. Este método es útil para depuración.

Parameters
messageEl mensaje que se desea imprimir en el log. Puede ser cualquier texto que ayude a entender el estado del programa.

Ejemplo de uso: 



public static void main(String[] args) {

    log("Este es un mensaje de prueba");

}

Salida esperada: 

[NombreClase:Linea] NombreMetodo: Este es un mensaje de prueba

Limitaciones: - Este método utiliza Thread.currentThread().getStackTrace(), lo que puede ser costoso si se llama frecuentemente en bucles o métodos muy utilizados. - No está diseñado para reemplazar un sistema de logging robusto como Log4j o SLF4J.
