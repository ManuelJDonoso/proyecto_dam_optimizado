Valida si un texto dado cumple con un formato específico de número de teléfono. 

El texto puede cumplir con uno de los siguientes formatos\+: 
\begin{DoxyItemize}
\item {\bfseries{Secuencia opcional\+:}} Comienza con {\ttfamily +} seguido de exactamente dos dígitos. 
\item Sin prefijo\+: 
\begin{DoxyItemize}
\item Nueve dígitos consecutivos ({\ttfamily \textbackslash{}d\{9\}}). 
\item Tres dígitos seguidos de un guion {\ttfamily -\/}, luego dos dígitos, otro guion {\ttfamily -\/}, dos dígitos más, otro guion {\ttfamily -\/}, y finalmente dos dígitos ({\ttfamily \textbackslash{}d\{3\}-\/\textbackslash{}d\{2\}-\/\textbackslash{}d\{2\}-\/\textbackslash{}d\{2\}}). 
\item Tres dígitos seguidos de un punto {\ttfamily .}, luego dos dígitos, otro punto {\ttfamily .}, dos dígitos más, otro punto {\ttfamily .}, y finalmente dos dígitos ({\ttfamily \textbackslash{}d\{3\}\textbackslash{}.\textbackslash{}d\{2\}\textbackslash{}.\textbackslash{}d\{2\}\textbackslash{}.\textbackslash{}d\{2\}}). 
\end{DoxyItemize}
\item Con prefijo ({\ttfamily +XX})\+: 
\begin{DoxyItemize}
\item Formato con guion\+: {\ttfamily +XX-\/\textbackslash{}d\{3\}-\/\textbackslash{}d\{2\}-\/\textbackslash{}d\{2\}-\/\textbackslash{}d\{2\}}. 
\item Formato con punto\+: {\ttfamily +XX.\textbackslash{}d\{3\}.\textbackslash{}d\{2\}.\textbackslash{}d\{2\}.\textbackslash{}d\{2\}}. 
\end{DoxyItemize}
